# OutputDatadogSeverity

Default value for message severity. When you send logs as JSON objects, the event's '__severity' field (if set) will override this value.


## Values

| Name        | Value       |
| ----------- | ----------- |
| `EMERGENCY` | emergency   |
| `ALERT`     | alert       |
| `CRITICAL`  | critical    |
| `ERROR`     | error       |
| `WARNING`   | warning     |
| `NOTICE`    | notice      |
| `INFO`      | info        |
| `DEBUG`     | debug       |
